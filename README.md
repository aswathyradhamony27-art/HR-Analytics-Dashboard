# HR-Analytics-Dashboard (Tableau)

 Project Overview
 
This HR Analytics Dashboard provides insights into employee data, attrition trends, and workforce demographics. It helps organizations understand employee behavior and improve retention strategies.

 Objectives
- Analyze employee attrition and retention
- Understand workforce demographics
- Identify high-risk departments
- Evaluate job satisfaction levels

 Key Metrics
-  Total Employees: 1,470
-  Attrition Count: 237
-  Attrition Rate: 16.12%
-  Active Employees: 1,233
-  Average Age: 37

 Dashboard Features

 1. Employee Overview
- Displays total employees, attrition count, and active workforce

 2. Attrition by Gender
- Compares male and female attrition levels

 3. Department-wise Attrition
- Pie chart showing attrition distribution across:
  - HR
  - R&D
  - Sales

 4. Age Group Analysis
- Histogram showing employee distribution across age groups

 5. Job Satisfaction Rating
- Matrix visualization of job satisfaction (1–4 scale)
- Segmented by job roles

 6. Education Field Analysis
- Attrition based on education backgrounds like:
  - Life Sciences
  - Medical
  - Marketing
  - Technical Degree

 7. Attrition by Age & Gender
- Donut charts showing attrition trends across different age groups and gender

 Tools & Technologies
- Power BI
- Data Modeling
- Data Visualization

 Insights
- Highest attrition observed in certain departments
- Younger employees show higher attrition rates
- Job satisfaction varies significantly by role
- Education background influences attrition trends
